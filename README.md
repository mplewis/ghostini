# Ghostini

Proxy content from your [Ghost blog](https://ghost.org/) into a
[Gemini site](https://gemini.circumlunar.space/).

# Environment Variables

- `GHOST_SITE`
- `CONTENT_KEY`
- `PER_PAGE`
